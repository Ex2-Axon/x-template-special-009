# Agents

## Overview

Apply the following edits to **every template folder** matching the pattern `x-template-special-*`. For each template folder, extract its numeric suffix and use that as `<TEMPLATE_ID>`. For example :

- For folder `x-template-special-001`, set `<TEMPLATE_ID>` to `001`
- For folder `x-template-special-042`, set `<TEMPLATE_ID>` to `042`

Repeat all steps below for each template folder.

---

## Step 0: Set Template ID Variable

Extract the template folder suffix. Example: if processing `x-template-special-001`, then `<TEMPLATE_ID>` = `001`.

---

## Step 1: Update Configuration Files for Folder `x-template-special-<TEMPLATE_ID>`

### 1a. Update vite.config.ts

**File:** `x-template-special-<TEMPLATE_ID>/vite.config.ts`

**Action:** Find the `base` property (do not rely on line numbers) and replace its value with:
```
'/x-template-special-<TEMPLATE_ID>/'
```

Example: if `<TEMPLATE_ID>` is `001`, the base should be `/x-template-special-001/`.

### 1b. Update .github/workflows/screenshot.yml (First Location)

**File:** `x-template-special-<TEMPLATE_ID>/.github/workflows/screenshot.yml`

**Action:** Search for the line containing `await page.goto('https://ex2-axon.github.io/x-template-special-...` (do not rely on line numbers). Replace the URL path segment with the actual template folder name:
```
await page.goto('https://ex2-axon.github.io/x-template-special-<TEMPLATE_ID>/'
```

### 1c. Update .github/workflows/screenshot.yml (Second Location)

**File:** `x-template-special-<TEMPLATE_ID>/.github/workflows/screenshot.yml`

**Action:** Search for the line containing `SITE_URL="https://ex2-axon.github.io/x-template-special-...` (do not rely on line numbers). Replace the URL path segment:
```
SITE_URL="https://ex2-axon.github.io/x-template-special-<TEMPLATE_ID>/"
```

---

## Step 2: Delete Screenshot Directory

**File:** `x-template-special-<TEMPLATE_ID>/.github/screenshots`

**Action:** Delete this directory because it is automatically regenerated when the GitHub Actions workflow runs. Commit the deletion to the repository:
```
git rm -r x-template-special-<TEMPLATE_ID>/.github/screenshots
git commit -m "Remove autogenerated screenshots directory for x-template-special-<TEMPLATE_ID>"
```

---

## Step 3: Copy and Rename Agent Script

**Source:** `x-template-special-<TEMPLATE_ID>/scripts/x-template-special.tsx`  
**Destination:** `x-template-special-<TEMPLATE_ID>/src/App.tsx`

### 3a. Delete Existing App.tsx

**File:** `x-template-special-<TEMPLATE_ID>/src/App.tsx`

**Action:** Delete this file from the repository.

**Error Handling:** If `App.tsx` does not exist in `x-template-special-<TEMPLATE_ID>/src/`, log a message (e.g., "App.tsx not found in x-template-special-<TEMPLATE_ID>/src") and continue to the next step without failing.

### 3b. Copy Script to src Folder

**Source File:** `x-template-special-<TEMPLATE_ID>/scripts/x-template-special.tsx`

**Action:** Copy this file to `x-template-special-<TEMPLATE_ID>/src/` (same template folder as source, NOT to x-template-special-001).

**Error Handling:** If the source file does not exist or a file named `App.tsx` already exists in the destination, back up the existing file by renaming it to `App.tsx.bak` before proceeding. Include this backup in a git commit with message: "Backup existing files for x-template-special-<TEMPLATE_ID> before updating App.tsx".

### 3c. Rename File to App.tsx

**File:** The copied file in `x-template-special-<TEMPLATE_ID>/src/`

**Action:** Rename `x-template-special.tsx` to `App.tsx` in the destination folder `x-template-special-<TEMPLATE_ID>/src/`.

---

## Notes

- All file operations should be committed to the repository with descriptive commit messages.
- Perform these steps for **all** folders matching `x-template-special-*` (e.g., 001, 002, 003, ..., 010, 042, etc.), not just for x-template-special-001.
- Use the actual folder suffix as `<TEMPLATE_ID>` for each template to ensure consistency across all files in that template.
